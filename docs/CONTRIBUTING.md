# Contributing to Prepify

Welcome to the Prepify project! Follow these steps to get started.

## Onboarding
1. **Read the Architecture Guide**: Understand how data flows through the app.
2. **Explore the Theme**: Familiarize yourself with `AppTheme` and how to use tokens.
3. **Navigation**: Learn how to use Navigation 3 in `AppNavigation.kt`.

## Development Process
1. **Pick a Task**: Ensure it aligns with our current milestones.
2. **Create a Feature Branch**: `feature/[short-description]`.
3. **Follow the UI Guide**: Use the workflow defined in `UI_DEVELOPMENT_GUIDE.md`.
4. **Write Tests**: Ensure domain logic and ViewModels are covered.
5. **Linting**: Run `./gradlew ktlintCheck` before committing.

## Pull Request Guidelines
- Include screenshots/videos for UI changes.
- Ensure all CI checks pass.
- Update documentation if you introduce new architecture patterns or components.
